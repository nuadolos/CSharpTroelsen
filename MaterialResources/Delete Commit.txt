Для удаления коммита:
	1) Зайти в Visual Studio (открыв проект);
	2) Нажать вкладку "Git" и выбрать "Просмотреть журнал ветвей";
	3) Нажать ПКМ по предыдущему коммиту и выбрать "Сброс -> Удалить изменения";
		Примечание к 3 пункту: "Сохранить изменения" лучше изучить функцию,
			Также скопировать файлы на рабочий стол, чтобы гарантировать безопасность файлов.
			"Сохранить изменения" - удаляет коммиты, но оставляет все файлы и его именения на месте
	4) Открыть "Git Bush" в папке проекта
	5) Добавить файлы и коммит;
	6) Запушить изменения с модификатором -f:
		git push -f origin <имя_ветки> (В случае данного проекта master)

Все коммиты до сброса будут перетерты.

Безопасная версия удаления коммитов на производстве:
	git push --force-with-lease origin <имя_ветки>
